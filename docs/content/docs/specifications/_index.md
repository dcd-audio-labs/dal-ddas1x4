# Specifications

Technical specifications and reference data for the SPDIF 1x4 Active Splitter.

## Quick Specs

| Parameter | Value |
|-----------|-------|
| Input | 1x SPDIF coaxial (RCA), 75 ohm terminated |
| Outputs | 4x SPDIF coaxial (RCA), 75 ohm source impedance |
| Buffer IC | TI THS7374IPWR (TSSOP-14) |
| Voltage Gain | 6 dB (2x) |
| Bandwidth | ~400 MHz (LPF bypassed) |
| Power Source | USB 5V (4.75-5.25V) |
| Current Draw | ~50 mA typical, < 100 mA max |
| Sample Rates | 32-192 kHz PCM, compressed bitstreams |
| Cable Support | Up to 30m RG-6, up to 30m RG-59 (RG-6 preferred > 20m) |
| PCB Size | ~60 x 40 mm |
| Schematic | Rev 1.0 (2025-02-22), KiCad 9.0 |
